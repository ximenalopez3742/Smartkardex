# 🎓 Smartkardex — Sistema de Análisis Académico UDG

Aplicación web para analizar kárdex de la Universidad de Guadalajara (IELC/ICOM).  
Sube un PDF de kárdex y obtén: materias disponibles, bloqueadas, alertas del Art. 33 y progreso académico.

---

## ✨ Funciones

- 📄 Carga de kárdex en PDF (drag & drop)
- 📊 Dashboard con créditos, promedio y avance por área
- ✅ Materias disponibles para el próximo ciclo
- 🔒 Materias bloqueadas con el prerrequisito que falta
- ⚠️ Alertas: Artículo 33, Servicio Social, Prácticas Profesionales

---

## 🚀 Correr localmente

```bash
pip install -r requirements.txt
python app.py
```

Abre: http://localhost:5000
